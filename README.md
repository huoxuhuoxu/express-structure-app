#### 生成nodejs-express项目: 项目构建命令行工具

##### install
	
	sudo npm install express-structure-app -g
	
##### using, 命令行运行

	express-structure-app: 项目默认初始化
	express-structure-app [project-name]: 给项目命名
	
##### rely on
	
	git
	nodeJs version > 7.7.8
